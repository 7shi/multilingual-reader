## 実行

```bash
bash batch.sh
```

スクリプトは翻訳→評価→集約の3フェーズに分離している。翻訳モデルと評価モデル（qwen3.6）を連続稼働させることでモデル切り替えのオーバーヘッドを避けるため、全モデルの翻訳をまとめて実行してから評価に移る。
