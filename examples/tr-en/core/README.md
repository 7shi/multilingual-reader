# examples/tr-en/core/

英語原文からコア言語（ドイツ語、日本語、中国語）への翻訳・評価を行います。

## 実行

`make` で翻訳・評価・集計を一括実行。翻訳は `tr/`、評価は `evals/`、スコアは `SCORES.txt` に出力。

- 翻訳モデル: gemma4:26b
- 評価モデル: qwen3.6
- 設定: threshold=20・keep=5・CoT なし・用語ファイル注入（`../../terms/*-en.{json,tsv}`）
- 既存ファイルはスキップされるため途中再開可能

## 翻訳品質の概要

評価結果（`SCORES.txt`）および内容の検証（評価者: `qwen3.6`）に基づく品質傾向は以下の通りです。学習リソースが豊富なコア言語においては、極めて安定した高品質な翻訳（軒並み90点台後半）が出力されています。

**高品質〜出版レベル (90点以上):** ドイツ語 (de), 日本語 (ja), 中国語 (zh)

- **専門用語と文脈の適応**: AIや量子力学の複雑な専門用語を各言語の標準的な学術・業界用語に正確にマッピングしつつ、ポッドキャスト特有の「カジュアルで分かりやすい会話のトーン」を完璧に再現しています。
- **ネイティブレベルの流暢さ**: 相槌や文のつながりが自然であり、翻訳特有の不自然さ（翻訳語）がほとんど見られません。そのままメディア公開できる出版レベル（publication-ready）の品質に達していると評価されています。

例外として、ドイツ語の `momentum` トピックのみ 88点 とわずかにスコアを落としました。これは、内容の正確さや論理展開は完璧であるものの、「Pitcher-Hügel（ピッチャーマウンド）」「Peak（ピーク）」「neu verdrahten（rewireの直訳）」といった英語の直訳（カルク）やアングリシズムが散見され、母語話者にとっての「流暢さ（fluency）」の項目で軽微な減点を受けたためです。実用上の問題はありません。
