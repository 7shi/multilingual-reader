# examples/tr-en/

英語原文からドイツ語・日本語・中国語への翻訳を行い、`examples/` の参照訳を更新するためのディレクトリ。

## 前提

- `examples/terms/*-en.{json,tsv}` が最新の英語ファイルをもとに再生成・校正済みであること（`examples/terms/batch.sh` で生成）

## 実行

`make` で翻訳・評価・集計を一括実行。翻訳は `tr/`、評価は `evals/`、スコアは `SCORES.txt` に出力。

- 翻訳モデル: gemma4-26b
- 設定: threshold=20・keep=5・CoT なし・用語ファイル注入（`../terms/*-en.{json,tsv}`）
- 既存ファイルはスキップされるため途中再開可能
- threshold=20 は 26B モデルで問題ないことを確認済み（tr-fr の threshold=10 より広いコンテキストを保持）

## 翻訳品質の概要

ベース言語はドイツ語 (de)、日本語 (ja)、中国語 (zh)。

onde のみ以下の言語が追加:

エスペラント (eo), ヒンディー語 (hi), テルグ語 (te), カンナダ語 (kn), トルコ語 (tr), エストニア語 (et), セルビア語 (sr)

評価結果（`SCORES.txt`）および内容の検証に基づく各言語の品質傾向は以下の通りです。

**高品質 (90点以上):** トルコ語 (tr), セルビア語 (sr)

構文、専門用語ともに正確で、自然な表現が維持されています。ほとんど修正なしで利用可能なレベルです。

**中〜低品質 (50点〜80点未満):** ヒンディー語 (hi), テルグ語 (te), カンナダ語 (kn)

文法的な誤りや綴りミスが散見されます。特に物理学の専門用語（例：「振幅」が一般的な「広がり」と訳されるなど）の不正確さや、翻字の不統一があるため、専門家による校正が必須です。

**致命的な欠陥 (50点未満):** エスペラント (eo), エストニア語 (et)

他言語（英語、韓国語、イタリア語など）の混入、存在しない造語、文脈を無視した致命的な誤訳（例：光線を元素のラジウムと誤訳、保存則を食品の保存と誤訳）が多発しています。実用には耐えず、ゼロからの再翻訳を推奨します。
