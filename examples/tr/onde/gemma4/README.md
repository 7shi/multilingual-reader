# examples/tr/onde/gemma4/

onde テキストを対象とした追加言語を含む翻訳・評価を行います。

対象言語: [common.mk](../../common.mk) で定義された `CORE_LANGS` + `EXTRA_LANGS`

## 実行

`make` で翻訳・評価・集計を一括実行。翻訳は `tr/`、評価は `evals/`、スコアは `SCORES.txt` に出力。

- 翻訳モデル: gemma4:26b
- 評価モデル: qwen3.6
- 設定: threshold=20・keep=5・CoT なし・用語ファイル注入（`../../terms/*-en.{json,tsv}`）

**高品質 (90点以上):** フランス語 (fr), 日本語 (ja), ロシア語 (ru), アラビア語 (ar), 中国語 (zh), ペルシア語 (fa), フィンランド語 (fi), 朝鮮語 (ko), ドイツ語 (de), ウクライナ語 (uk), スペイン語 (es), セルビア語 (sr), イタリア語 (it), ポルトガル語 (pt), ポーランド語 (pl), トルコ語 (tr)

構文、専門用語ともに正確で、自然な表現が維持されています。アラビア語やペルシア語など、右横書き（RTL）の言語でも極めて安定しており、ほとんど修正なしで利用可能なレベルです。ロシア語が 97 点で最良クラス、イタリア語・ポルトガル語・ウクライナ語・ポーランド語も軽微な語法修正だけで実用可能でした。

**実用範囲 (80〜89点):** ヘブライ語 (he), スウェーデン語 (sv), インドネシア語 (id), ブルガリア語 (bg)

専門用語は概ね正確ですが、一部に軽微な文法ミス（格の誤りなど）や語彙の不適切な選択（例：「発散」の誤訳など）が含まれており、軽い推敲が必要です。ヘブライ語はセム語としての構造は維持していますが、話者タグの崩れや用語（psiの翻字など）の不一致が目立ちます。スウェーデン語は直訳気味の語法が散見されるものの情報保持は良好です。インドネシア語は英語の直訳調（calque）が残る箇所（例：psi kuadrat）が見られますが、理解に支障はありません。

**中品質 (60〜79点):** チェコ語 (cs), オランダ語 (nl), ヒンディー語 (hi), ウルドゥー語 (ur), テルグ語 (te)

文法的な誤りや綴りミスが散見されます。チェコ語とオランダ語は内容自体は正確ですが、英語の直訳に引きずられた不自然な言い回しが残り、放送・出版前にネイティブ校正が必要です。ウルドゥー語は確率(probability)を安全(امان)と誤訳するなどの語彙のミスが見られます。特に物理学の専門用語（例：「振幅」が一般的な「広がり」と訳されるなど）の不正確さや、翻字の不統一があるため、専門家による校正が必須です。

**致命的な欠陥 (60点未満):** タミル語 (ta), カンナダ語 (kn), ハンガリー語 (hu), エスペラント (eo), エストニア語 (et), インターリングア (ia)

他言語（英語、朝鮮語、ルーマニア語、ベトナム語、イタリア語など）のランダムな混入（コードスイッチング）、存在しない造語、文脈を無視した致命的な誤訳（例：光線を元素のラジウムと誤訳、保存則を食品の保存と誤訳）が多発しています。インターリングアではアラビア語 `ما`、ポルトガル語 `com um`、英語 `pattern` などが混入し、統語も崩壊しました。タミル語は英語（psi, Magnificent）の混入や綴りミスが深刻です。実用には耐えず、ゼロからの再翻訳を推奨します。
