# examples/tr/onde/gemma4/

onde テキストを対象とした追加言語を含む翻訳・評価を行います。

対象言語: [common.mk](../../common.mk) で定義された `CORE_LANGS` + `EXTRA_LANGS`

## 実行

`make` で翻訳・評価・集計を一括実行。翻訳は `tr/`、評価は `evals/`、スコアは `SCORES.txt` に出力。

- 翻訳モデル: gemma4:26b
- 評価モデル: qwen3.6
- 設定: threshold=20・keep=5・CoT なし・用語ファイル注入（`../../terms/*-en.{json,tsv}`）

## 翻訳品質の概要

評価結果（`SCORES.txt`）および内容の検証に基づく各言語の品質傾向は以下の通りです。

| :--- | :--- |

| フィンランド語 (fi) | 95 |
| 朝鮮語 (ko) | 95 |
| セルビア語 (sr) | 93 |
| トルコ語 (tr) | 91 |
| インドネシア語 (id) | 86 |
| ブルガリア語 (bg) | 80 |

| エストニア語 (et) | 22 |

**高品質 (90点以上):** アラビア語 (ar), フィンランド語 (fi), 朝鮮語 (ko), トルコ語 (tr), セルビア語 (sr)

構文、専門用語ともに正確で、自然な表現が維持されています。アラビア語は非連結的な形態論（語根ベースの屈折）を持つ複雑な言語構造ですが、gemma4では極めて安定しており、ほとんど修正なしで利用可能なレベルです。

**実用範囲 (80〜89点):** インドネシア語 (id), ブルガリア語 (bg)

専門用語は概ね正確ですが、一部に軽微な文法ミス（格の誤りなど）や語彙の不適切な選択（例：「発散」の誤訳など）が含まれており、軽い推敲が必要です。インドネシア語は英語の直訳調（calque）が残る箇所（例：psi kuadrat）が見られますが、理解に支障はありません。

**中品質 (60〜79点):** ヒンディー語 (hi), テルグ語 (te)

文法的な誤りや綴りミスが散見されます。特に物理学の専門用語（例：「振幅」が一般的な「広がり」と訳されるなど）の不正確さや、翻字の不統一があるため、専門家による校正が必須です。

**致命的な欠陥 (60点未満):** タミル語 (ta), カンナダ語 (kn), ハンガリー語 (hu), エスペラント (eo), エストニア語 (et)

他言語（英語、朝鮮語、ルーマニア語、ベトナム語、イタリア語など）のランダムな混入（コードスイッチング）、存在しない造語、文脈を無視した致命的な誤訳（例：光線を元素のラジウムと誤訳、保存則を食品の保存と誤訳）が多発しています。タミル語は英語（psi, Magnificent）の混入や綴りミスが深刻です。実用には耐えず、ゼロからの再翻訳を推奨します。
